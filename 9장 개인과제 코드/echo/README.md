# echo

간단한 Echo 서버/클라이언트 예제

- `echo_server.py` : 서버
- `echo_client.py` : 클라이언트

로컬 테스트 방법

1. 서버 실행: `python "c:\np-202110152\9장 과제\echo\echo_server.py"`
2. 새 터미널에서 클라이언트 실행: `python "c:\np-202110152\9장 과제\echo\echo_client.py"`

원격(GitHub)에 올리려면:

- GitHub 웹에서 새 리포지토리를 만들고 remote를 추가한 뒤 push 하거나,
- GitHub CLI(gh)가 설치되어 있으면 `gh repo create`로 자동 생성 후 push
